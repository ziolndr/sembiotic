# Sembiotic real scientific media catalog

Sembiotic does not generate scientific result imagery. Search records use, in order:

1. imagery attached to the indexed source record;
2. source-page Open Graph imagery captured during ingestion;
3. a deterministic match against the credited real microscopy catalog below.

The fallback catalog is used only as visual context for a ranked object. It is not presented as evidence produced by that object.
